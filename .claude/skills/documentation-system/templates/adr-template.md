## Notes

**How to use this template:**
1. Copy to `decisions/NNN-decision-name.md` (number sequentially)
2. Fill in all sections (remove optional sections if not needed)
3. Update `decisions/README.md` index
4. Link from relevant CLAUDE.md files

**When to create an ADR:**
- Choosing between architectural approaches
- Adopting new technologies or frameworks
- Changing core conventions or patterns
- Making decisions that affect multiple components
- Any decision you might need to justify later

**Keep it concise but complete:**
- Target: 500-1000 words
- Focus on "why" not "how"
- Include enough detail to understand the decision later
- Be honest about trade-offs
